# Project Objectives

## Purpose

Build a lightweight, reproducible Kubernetes observability lab using WSL2 and KinD.

The primary goal is not simply to install Prometheus and Grafana, but to understand, measure, document and optimise an observability platform through controlled engineering experiments.

## Objectives

- Build a reproducible lab from scratch.
- Document every installation and configuration step.
- Establish measurable performance baselines.
- Change one variable at a time.
- Measure and document the results.
- Explain the engineering rationale behind every decision.

## Target Audience

- Kubernetes learners
- DevOps engineers
- Platform engineers
- Anyone interested in observability on constrained hardware

## Non-Goals

- Production deployment
- High availability
- Enterprise-scale monitoring

This repository focuses on understanding systems rather than deploying production infrastructure.
